## Overview

This application provides a comprehensive platform for interacting with an AI assistant. It supports rich text formatting, image uploads, and contextual memory to enhance the conversation experience. Additionally, it features real-time collaborative document editing and can be installed as a Chrome extension for webpage analysis functionality.

## Features

### Core Capabilities
- Interactive chat interface with AI responses
- Conversation history management (reset, download, upload)
- Follow-up question generation
- Rich text formatting with Markdown support
- Image upload for visual conversations
- Contextual memory management system

### User Experience
- Responsive design using Bootstrap
- Adaptive UI elements with responsive button layouts
- Avatar selection and customization
- User preference settings

### Collaboration & Extension
- Real-time collaborative Markdown document editing
- Chrome extension with webpage analysis
- Content extraction and processing
- Seamless integration with Chrome's storage API

## Getting Started

### Prerequisites

- Node.js (v14.0.0 or higher)
- npm (v6.0.0 or higher)

### Installation

```bash
# Clone the repository
git clone https://github.com/deltamaster/react-tutorial-solutions.git
cd react-tutorial-solutions

# Install dependencies
npm install
```

### Development

Start the development server:

```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000) in your default browser. The page will automatically reload when changes are made to the source code.

## User Guide

### Basic Usage
1. Enter your Subscription Key in Settings before using the service
2. Type your question in the text input field and press Enter or click Send
3. View the AI's response in the conversation history
4. Use the follow-up questions generated by the AI for quick follow-up queries

### Conversation Management
Use the buttons in the top-right corner to manage your conversations:

| Action | Description |
|--------|-------------|
| Reset Conversation | Clears the current chat history |
| Download History | Saves the conversation as a JSON file |
| Upload History | Loads a conversation from a JSON file |

### Advanced Features

#### Memory Management
- Access the Memory tab to view, add, edit, or delete contextual memory entries
- Memory entries are stored in your browser's localStorage
- Use this feature to provide persistent context for your conversations

#### Collaborative Editing
- Navigate to the "Co-Edit" tab to use the collaborative Markdown editor
- See real-time preview of your Markdown formatting
- Changes are automatically synchronized with other users
- Document content is loaded when the editor becomes visible for better performance

#### Settings Customization
- Access the Settings tab to customize your experience
- Select or upload a user avatar
- Configure application preferences

## Chrome Extension

### Chrome Extension Installation

1. Build the application:
```bash
npm run build
```

2. Install in Chrome:
   - Open Chrome and navigate to `chrome://extensions/`
   - Enable "Developer mode" in the top right corner
   - Click "Load unpacked" and select the `build` directory
   - The extension will be installed in your extensions toolbar

### Extension-Specific Features

- **Right-click menu integration**: Right-click on any webpage and select "Analyze Page"
- **Content analysis**: Extracts and processes webpage content, removing unnecessary elements
- **Popup interface**: Full application access through the extension popup
- **Storage integration**: Uses Chrome's local storage for data persistence

### Key Components

| Component | Description |
|-----------|-------------|
| AppContent | Main component managing chatbot state and tabs navigation |
| ConversationHistory | Displays chat history between user and AI |
| QuestionInput | Handles user input for questions and image uploads |
| ApiKeyInput | Manages the Gemini API key input |
| FollowUpQuestions | Shows AI-generated follow-up questions |
| Memory | Interface for managing contextual memory data |
| MarkdownEditor | Real-time markdown editor with preview |
| Settings | User settings management interface |
| AvatarSettings | Component for selecting and customizing avatars |

Create an optimized production build:

```bash
npm run build
```

This will generate an optimized build in the `build` directory ready for deployment.

## Important Notes

- A valid Subscription Key is required to use the AI service (configure in Settings)
- For Chrome extension functionality, site permissions may be required
- Image upload supports PNG, JPEG, WEBP, HEIC, and HEIF formats with a 20MB size limit
